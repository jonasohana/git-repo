p>After making and saving that change, and refreshing the page in the browser, the page still shows three buttons. But clicking Button 1 or Button 3 has no effect, because now the #button2 selector in the jQuery code links the code directly to the one and only button that contains id="button2".

One of the beauties of jQuery is that you get the best of both worlds with selectors. You can easily write a jQuery function that applies to any one element on the page. Or you can write it to apply to many elements. And the selector syntax that you use to target the function is the same as the selector syntax that you use for targeting CSS style rules. Fun With jQuery Special Effects

The ability to apply JavaScript in much the same way you can apply CSS is one of jQuery's best features and part of the reason for its widespread success. But that's not the whole story. As you'll learn in this chapter, in addition to being able to execute any JavaScript you want within jQuery, you can also use some of its built-in effects. Some are similar to CSS3 transition effects. But they have one advantage over CSS3 transitions. Unlike CSS3 transitions, jQuery effects are written to work even with older Web browsers, not just the very newest browsers that have CSS3 support. Here is a quick overview of some of the most popular effects.

Effect Description

.fadeToggle Switches between opaque and invisible by fading
.hide() Hides the selected elements

.slideDown() Reveals selected element by sliding down
.slideToggle() Shows or hides the selected element by sliding up or down
.slideUp() Hides the selected element by sliding up
